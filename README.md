# vektor_if

VektorIF é uma aplicação Flutter para gerenciamento e visualização de mapas internos de instituições.

Para documentação detalhada sobre o projeto, incluindo funcionalidades, estrutura e modelos de dados, consulte [DOCUMENTATION.md](DOCUMENTATION.md).
